CSS3 circular image. You can basically use an image of any dimensions,the drawback would be that it won't be focused correctly compared to that of a properly sized image to be rounded.
CSS
          .circ-img {
            width: 300px;
            height: 300px;
            border-radius: 150px;
            -webkit-border-radius: 150px;
            -moz-border-radius: 150px;
            background: url('image-url.png') no-repeat;
          }
          /* Drag Off Image */
          .circ-img img{
            opacity: 0;
            filter: alpha(opacity=0);
          }
